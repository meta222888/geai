# Geai Roadmap

## v0.1

- Native Win32 chat window
- Gemini generateContent API
- API key / API base / model settings
- HTTP proxy setting
- JSONL session save
- Deno Deploy proxy

## v0.2

- Streaming output
- Conversation loading UI
- Markdown rendering
- Better JSON parser
- Dark mode

## v0.3

- Local encrypted API key storage with Windows DPAPI
- System prompt templates
- Export Markdown / JSON
- Tray mode
- Global hotkey

## Design Principles

- Keep binary small
- Avoid Electron / Qt
- Prefer Windows native APIs
- Make proxy optional
- Keep user data local
